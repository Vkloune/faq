# Горячие клавиши
### Копировать:
 Ctrl+C
### Вставить:
Ctrl+V
### Сохранить: 
Ctrl+S
### Вырезать: 
Ctrl+X
### Отмена:
Ctrl+Z
## Работа с git
 ### Скачать проект:
 ```bash
 info git clone <URL>
 ```
 ### Учесть все файлы:
  ```bash
git add .
```
### Сохранить коммит:
 ```bash
git commit -m "..."
```
### Отправка в облако:
 ```bash
git push
```
